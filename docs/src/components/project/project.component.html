is an official implementation of Micro Focus UX using AngularJS. This documentation contains a reference implementation and exmaples.

Installation

Manual Installation

Download ng-ias.css, ias-icons.css (NOTE: ias-icons.css requires the referenced font files as well), and ng-ias.js (NOTE: ias-icons.css requires the referenced font files. Add the stylesheets to your .html file:


                <link rel="stylesheet" href="ias-icons.css" />
                <link rel="stylesheet" href="ng-ias.css" />
                <script src="ng-ias.js"></script>

NPM


                npm install --save --save-exact git@github.com:jedwardhawkins/ng-ias
