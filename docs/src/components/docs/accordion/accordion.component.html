Different start item

By default, [toggle-mutex-groups] show the first component on start. Use the [toggle-start-open] directive to toggle a specific item on start. If toggle-start-open is used on multiple accordion items, the last one declared will be used.

[toggle-group]

[toggle-mutex-groups require that one and only one component is showing at all times and are the IAS preferred method for accordions. Use the [toggle-group] attribute instead of [toggle-mutex-group] to allow multiple accordion items to show or collapse simultaneously.

Open/Close all

When a [toggle-group] has many accordion items, it may be desirable to provide a way to open or close all items simultaneously. The IasToggleService provides additional methods to facilitate this interaction.
