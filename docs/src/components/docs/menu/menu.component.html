Positioning w/ [mf-align]

The mf-align attribute has the form mf-align="[H_ALIGN] [V_ALIGN]?" where H_ALIGN is one of [start, center, end] and V_ALIGN is one of [top, center, bottom] and V_ALIGN is optional. The default is mf-align="start top". start and end depend on the layout of the containing element. The default and when nested under [dir="ltr"], start is on the left and end is on the right. When nested under [dir="rtl"], start is on the right and end is on the left.
