Input

Input components cover the functionality and styling of text boxes, password text boxes, text areas, select lists, radio buttons, checkboxes, and an integer only text box.

Text field with icon

Text fields with icons are popular for search fields and any other field which requires a pictorial differentiator. The icon field uses the same reference to the icon font as the ias-icon component.

Text area

Resizing text area

Resizing text areas will grow and shrink based on the contents in the text area. When you add more contents the text area will grow to show all of the contents without scrolling. When you remove some contents the text area will shrink down to the right size.

Integer only input

Integer only input fields show an error style if the value isn't a valid number. The input field also supports a min and max attribute to set the allowable minimum and maximum values.
