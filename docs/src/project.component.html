Installation

Manual Installation

Download mfux.css, mf-icons.css (NOTE: mf-icons.css requires the referenced font files as well), and ng-mfux.js (NOTE: mf-icons.css requires the referenced . Add the stylesheets to your .html file:

  • <link rel="stylesheet" href="mfux.css" />
  • <link rel="stylesheet" href="mf-icons.css" />
  • <script src="ng-mfux.js"></script>

NPM

npm install --save --save-exact git@github.com:jedwardhawkins/ng-mfux
